Evaluate a simple expressionThe code pad can be used to evaluate simple expressions.Expressions can refer to standard Java values and objects, as well as classes fromthe current project. The code pad will display the result value, followed by itstype (in parenthesis), or an error message if the expression is incorrect.Examples are:4 + 45"hello".length()Math.max(33, 4)(int) 33.7javax.swing.JOptionPane.showInputDialog(null, "Name:")
