Open a classmain window:context menu: Open Editor"Open a class" is the expression we commonly use to shorten the more precise"Open an editor to show the source of a class". You open a class if you- want to see the interface of the class- want to see the implementation- want to edit the class (change the interface or implementation)You can also open a class by double clicking the class icon.Depending on which view you used when you last looked at the class, you mightsee it in implementation view or interface view.
