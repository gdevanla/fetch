
UC49 Patient Education Use Case        

An HCP is a registered user of the iTrust Medical Records system (UC2). A Patient is a registered user of the iTrust Medical Records system(UC1). The iTrust user has authenticated himself or herself in the iTrust Medical Records system (UC3).

An admin can enter new diagnosis codes, along with URLs to pages containing more information about the diagnosis. A patient will be presented with the link for more information whenever that diagnosis code is displayed in their information .

   : An admin or HCP can view the diagnosis codes currently in the database, along with the URL associated with a particular diagnosis code.
   : An admin or HCP can add/edit a URL to a diagnosis code.
   : Any time a patient is presented with a diagnosis code, they should also be presented with a link to a page containing more information about the diagnosis. See also UC9, UC10, UC15, UC21, UC45.

 Any edits to the diagnosis/URL should be logged.</textarea>
